# 命令
- 输入 `GeekHologram` 或 `hd` 可查看详细命令

```
使用: /hd
        ├── create [<ID>] [<save>] # 创建全息图
        ├── delete [<ID>] [<目标玩家>] # 删除全息图
        ├── teleport [<ID>] # 传送到指定全息图
        ├── moveHere [<ID>] # 移动全息图到你位置
        ├── list # 移查看所有全息图
        ├── operate [<ID>] # 对全息图进行编辑
        └── reload # 重载配置
```

## 权限
```
geekhologram.command.create
geekhologram.command.delete
geekhologram.command.teleport
geekhologram.command.moveHere
geekhologram.command.list
geekhologram.command.operate
geekhologram.command.reload
```

## 玩家命令&权限 - 2023-9-25更新

### 命令
```
使用: /phd
        ├── create [<ID>] [<第一行>] # 创建全息图
        ├── remove [<ID>] # 删除全息图
        ├── add [<ID>] # 添加一行
        ├── del [<ID>] # 删除一行
        └── edit # 快捷编辑器
```

### 权限
```
hologram.player.command.create
hologram.player.command.edit
hologram.player.command.add
hologram.player.command.del
hologram.player.command.remove
```
