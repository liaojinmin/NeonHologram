确保你已经有正规途径取得的插件 JAR 后，进行安装

### 关于依赖

GeekHologram，无需任何**第三方插件**作为前置硬性依赖

如果你需要解析变量 请安装 PlaceholderAPI

在启动时会需要联网从阿里云中央仓库下载依赖，所有依赖均为知名开源项目  
下载后的依赖全部会被缓存到服务端 `libs` 根目录

:::info
GeekHologram 的启动和完整插件功能的体验都需要联网环境
:::
